# 重要提示

## 保护财产安全

每年入学前后，都可能有社会人员通过微信公众号、微信/QQ群、小红书等公开平台主动联系新生，推销电话卡等实体物体或者贩卖焦虑，并从中获取较高回扣（应该不回扣焦虑）。面对陌生人的私聊、群聊邀请或所谓“新生专属优惠”，请先核实对方身份，不要随意提供姓名、证件、手机号、录取信息等个人资料，也不要在未经确认的情况下付款。

### 校园电话卡并非必需

办理银行卡时，经常有神秘电话卡推销人员声称必须使用宁波本地手机号码。一般来说，能正常接收验证码、由本人实名登记的中国大陆手机号码就已经足够了。当然，具体要求仍应以办理银行的官方说明为准，不要仅凭推销人员的说法作出决定。

同时校园电话卡也不会为选课提供特殊通道或更高优先级。选课结果主要受系统开放时间、访问人数和系统状态等因素影响（说人话： 看运气），是否办理校园电话卡不会影响正常的校园学习与生活。是否更换套餐，应根据自己的通信需求和运营商正式资费独立判断。以及需要注意，很多电话卡都是四年的合约期，在你毕业过后价格暴增，不要被坑了。

### 警惕诈骗

去年有大一新生因为新办银行卡信息尚未完善，收到了骗子通过虚拟号码发送的伪“银行动账提醒”短信。次日，骗子通过钉钉添加该同学，并引导其加入在线会议，以“教学操作”为名进行指导。为骗取信任，骗子首先要求该同学进行一笔小额转账（非直接转至银行卡账户），随后便将款项退还，制造出“流程正规”的假象。随后，对方以“账户存在限额，需要修改”为由，诱导该同学多次进行小额转账。最终，这些资金被骗子通过游戏充值和虚假账号交易等渠道迅速洗走，造成财产损失。

遇到类似情况，请务必做到：

- 对自称银行、客服、运营商或学校工作人员的短信和电话，通过官方网站或官方客服电话另行核实；
- 不加入陌生人发起的“教学会议”，不按对方要求开启屏幕共享或远程控制；
- 不相信“小额转账测试”“先转后退”“解除限额”“账户验证”等说辞；
- 不向陌生人转账，不提供验证码、银行卡密码、身份证照片或账户信息；
- 一旦发现异常，立即停止操作，联系银行冻结相关账户，并及时报警。

!!! warning "涉及金钱和个人信息时，请先停下来核实"
    凡是涉及转账、商品推销或索取个人信息的私聊和群聊，都不要仅凭对方自称的身份相信或操作。对于声称由学校或学生组织安排的商业活动，请通过学校官方渠道核实；未经证实，不要泄露个人信息，也不要贸然付款。
