# 转专业政策

转专业是从当前专业或专业类申请调整至另一专业或专业类。它与 2+2 / 4+0 学制选择、专业大类内部的方向分流不是同一件事，办理时间和规则也可能不同。

## 2026 级第一次转专业

2026 级本科新生入学后，可在一年级第一学期末的指定时段内提交专业 / 专业类调整申请。

1. 每名申请者最多可填报两个志愿。
2. 学校将结合目标专业的转专业名额和考核标准进行综合审核。
3. 审核结果于一年级第二学期初统一公布。
4. 转专业获批后，学生自第二学期起正式转入新专业 / 专业类就读。

如果目标专业的考核需要参考一年级两门科目的加权平均分，这里的“两门科目”指 RWAC（写作课）和 OCSa（口语课），计算方式为 RWAC 成绩占 $\frac{2}{3}$、OCSa 成绩占 $\frac{1}{3}$。

不同目标专业可能对一年级课程成绩、学院测试、作品集或其他条件有不同要求。下图可用于了解院系要求的常见形式；2026 级实际开放专业、名额和考核标准，应以入学后发布的《转专业政策及指南》为准。

## 第二次转专业

学校后续还会开放第二次转专业机会。具体申请对象、时间、名额、考核方式和转入年级等规则，需要等待学校后续通知。

!!! info "这几件事不要混在一起"
    简单来说，转专业、学制选择和大类分流是三套不同的流程：

    - **转专业：** 从现在的专业申请转到另一个专业，具体申请时间和操作方法看学校发布的《转专业政策及指南》。
    - **学制选择：** 在本专业提供的 4+0 和 2+2 之间作出选择。
    - **大类分流：** 已被某个专业大类录取后，再选择大类内的具体专业或方向。

    后两项通常由学院另行通知，不需要通过转专业流程办理。详见[学制选择与专业分流](pathways.md){ .resource-link }。
