# 网络与 IT

## 校园 Wi-Fi

校内主要有两个无线网络：

| Wi-Fi | 主要覆盖区域 | 说明 |
| --- | --- | --- |
| **eduroam** | 教学区、体育馆等 | 校园教学网络 |
| **UNNC-living** | 宿舍楼内 | 生活区网络，据悉后续将升级，目前先按现有方式连接 |

校园无线网络目前还没有做到室外区域 100% 全覆盖，但教学楼、宿舍楼、体育馆等室内区域已经覆盖。不同位置的信号强度可能有所差异。

## eduroam 是什么

eduroam 是全球教育与科研机构共同使用的无线网络。许多大学和科研机构都提供 eduroam；在宁诺连接成功后，前往其他加入 eduroam 的高校时，也可以继续使用自己的宁诺账号登录，不需要另外申请访客网络。

[查看 eduroam 官方介绍](https://eduroam.org/what-is-eduroam/){ .resource-link } · [查看 UNNC Wi-Fi 官方指南](https://www.nottingham.edu.cn/en/it-services/wi-fi.aspx){ .resource-link }

## 登录方法

连接 `eduroam` 或 `UNNC-living` 时，使用学校账号登录：

1. 用户名优先填写完整学校邮箱，例如 `用户名@nottingham.edu.cn`。
2. 密码填写学校账号密码。
3. 如果使用完整邮箱无法登录，可以尝试只填写邮箱中 `@nottingham.edu.cn` 前面的用户名。

如果最近修改过学校密码，设备可能仍保存着旧密码。可以先在 Wi-Fi 设置中“忽略 / 忘记”该网络，再重新选择网络并输入新密码。
