# 校园卡与宿舍入住

## 领取学生卡和学生证

完成现场注册和身份核验后，新生会领取学生卡及学生证。学生卡是校园内的重要身份与服务凭证，可用于宿舍门禁、校园消费及其他校内服务。

如属于经批准延期报到的新生，应先前往 The Hub（Portland Building 120）完成信息核验和证件领取。

## 开通宿舍门禁

现场报到时，学校会为学生卡开通本人宿舍楼和房间门禁。完成开通后，再持新生注册单前往所在宿舍楼前台办理入住。

!!! warning "先完成注册，再办理入住"
    只有已完成现场注册、网上选寝和住宿费缴纳的新生，才能按所选房源办理入住。实际房间以选寝结果和现场安排为准。

## 宿舍前台办理

在宿舍楼前台通常需要：

1. 出示新生注册单及本人证件；
2. 核对楼栋和房间信息；
3. 领取空调遥控器等房间物品；
4. 进入寝室后检查设施设备。

每个寝室首位办理入住的新生需领取一只空调遥控器。

## 入住后 24 小时内检查

进入房间后，应按照《2026 级学生公寓住宿手册》列出的设施设备逐项检查。如发现损坏、缺失或不符合说明，应在 **24 小时内**向宿舍楼前台报告；未在时限内报告，相关设施可能被视为入住时完好。

后续设施维修可通过后勤事务中心官方微信平台“宁诺小 UPay”提交报修申请。

## 宁诺小 UPay { #unnc-upay }

“宁诺小 UPay”是学校后勤和校园卡服务常用的微信公众号，建议入校后搜索并关注。常用功能包括：

- 寝室水电费查询、预充值和缴纳；
- 校园卡充值；
- 校园卡挂失；
- 宿舍设施报修及其他后勤服务。

!!! warning "校园卡丢失后及时挂失"
    如发现校园卡遗失，应尽快通过“宁诺小 UPay”办理挂失，避免卡内余额被他人使用。补卡地点、费用及办理流程请以 The Hub 的最新说明为准。

## 自备生活用品

学校不统一配备床上用品，新生应自行携带或就近购买。建议提前查看[物品准备](../preparation/checklist.md)和[宿舍房型说明](../campus-life/accommodation.md)。

校园卡问题可联系 The Hub：学校总机 `0574-88180000` 转分机 `8062`，邮箱 [thehub@nottingham.edu.cn](mailto:thehub@nottingham.edu.cn)。
