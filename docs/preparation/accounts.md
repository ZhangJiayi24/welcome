# 账号与邮箱

新生入学阶段会使用多个系统。它们名称相近，但用途不同：

| 系统 | 主要用途 | 2026 级使用时间 |
| --- | --- | --- |
| [IT Accounts](https://itaccounts.nottingham.ac.uk/) | 激活学校 IT 账号 | 8 月 11 日起 |
| 学校邮箱 | 接收注册、课程、报到和学院通知 | 账号激活后立即开始查看 |
| [NottinghamHub / MyNottingham](https://mynottingham.nottingham.ac.uk/) | 完成 NottinghamHub 注册，后续办理学生事务 | 8 月 11 日起 |
| [Application Hub](https://oaas.nottingham.edu.cn/#/new-student/login) | 填写个人信息、网上预注册、网上选寝、下载新生注册单、提交延期报到申请 | 8 月 11 日起按阶段开放 |
| [Moodle](https://moodle.nottingham.ac.uk/) | 访问学院衔接课和入学后的课程资料 | IT 账号激活后 |

## 第一步：激活 IT 账号

学校邮箱是新生与学校各部门沟通的重要工具。请按照录取邮箱收到的通知，前往 IT Accounts 网站激活账号，并尽快确认能够登录学校邮箱。

- 激活入口：[https://itaccounts.nottingham.ac.uk/](https://itaccounts.nottingham.ac.uk/)
- 账号激活问题：[itservicedesk@nottingham.edu.cn](mailto:itservicedesk@nottingham.edu.cn)

## 第二步：注册 NottinghamHub

账号激活后，学校会向学校邮箱发送 NottinghamHub 注册邀请。打开邮件中的邀请链接，使用收到的用户名和初始密码登录，并按页面提示完成注册。

- 入口：[https://mynottingham.nottingham.ac.uk/](https://mynottingham.nottingham.ac.uk/)
- 注册问题：[Academic-Service-Office@nottingham.edu.cn](mailto:Academic-Service-Office@nottingham.edu.cn)

## 第三步：填写 Application Hub 个人信息

请在 **2026 年 8 月 11 日至 9 月 3 日**期间登录 Application Hub 完成个人信息填写。只有完成个人信息填写的新生，才能在后续步骤中下载并打印新生注册单。

Application Hub 使用与学校邮箱相同的用户名和密码登录：

[打开 Application Hub](https://oaas.nottingham.edu.cn/#/new-student/login){ .resource-link }

如对新生个人信息填写有疑问，可联系 [NewStudentWelcome@nottingham.edu.cn](mailto:NewStudentWelcome@nottingham.edu.cn)。

!!! warning "保护账号"
    不要将账号密码、短信验证码或登录二维码发送给他人。学校不会要求你通过共享屏幕、远程控制或向私人账户转账来“激活”账号。
