# 新生防骗

## 谨防电信诈骗

新生入学前，可能收到各类短信、电话或网络消息，有人会冒充学校工作人员、银行客服或其他官方人员实施诈骗，包括但不限于谎称新生入学须额外缴纳“材料费”，否则无法报到、入学资格将被取消等。请警惕此类电话和短信，不要按照陌生人的要求转账、共享屏幕或提供验证码。一经发现，请及时通过学校正式邮件、届时建立的行政班群向辅导员核验，也可以联系学校招生录取办公室确认相关内容。

## 警惕各类“官方群”

学校相关通知通常会通过学校邮箱、行政班群、辅导员或其他明确告知的正式渠道发布。部分学生组织、招生组、校友会、同乡会或学生群体也可能建立民间新生群，但这些群组不等同于学校官方渠道。对于来路不明、声称自己是“UNNC官方新生群”“宁诺校园墙”或以学校名义招揽新生的个人和群组，请保持警惕，其中可能带有引流、推销或欺诈性盈利目的。建议尽可能通过学校正式通知、校内学生组织、学校招生组、校友会或同乡会建立的渠道获取信息，不要轻信身份不明的人员。如遇可疑情况，可要求负责人出示校园卡或学生证，并保留相关聊天和交易记录。

## 警惕不明人员

新生入学报到时，由于校园内人员流动较大，可能出现非校内人员推销生活用品、文具、手机卡等商品，甚至进入宿舍园区或宿舍内，带来安全隐患。同时，也可能有校内学生出于兼职需要代理各类商品，同学们可以根据个人实际需求独立判断。遇到身份不明的推销人员时，可以要求对方出示学生证或校园卡并留存必要证据；如发现疑似校外人员进入宿舍园区或存在持续骚扰、强行推销等情况，请及时联系保卫处或后勤事务中心处理。
