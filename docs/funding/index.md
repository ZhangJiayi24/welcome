# 奖助

宁诺的奖学金与资助来源包括学校、学院和教育发展基金会，不同项目分别关注学业表现、科研创新、社会实践、公益、体育、文艺、实习或创业能力。

奖项是否开放、申请对象、评选条件和时间每年都可能调整。准备申请时，应以学生门户、校内邮件、学院通知及负责部门发布的当年信息为准。

本模块将继续整理本科生常见的奖学金、竞赛资助、实践奖励及申请注意事项。
