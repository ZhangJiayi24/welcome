# 前言

承蒙遇到的老师们朋友们的指引和厚爱，我绝对不是传统意义上的好学生，甚至于高中不想读书还休了一年学，多亏了身边人的支持和鼓励，才让我考到这里。滴水之恩当以涌泉相报，作为“过来人”，我走过很多弯路，就不希望我的“后来人”也和我一样需要频繁碰壁，于是想要为他们指一条路，撑一把伞。

2025年暑假，我参与了学校在天津地区的招生，彼时我在上海交通大学做科研，不能线下参与，于是任起了线上微信群大总管（我自封），专门负责在各个微信群中答疑。在和新生、新生家长们的第一次接触中，我回忆起我报志愿时了解学校时遇到的一些困难，回忆起互联网上各种错综复杂的信息扑面而来时的无措——诚然我希望朋友们都有辨别信息的能力，但是需要培养——我觉得他们也需要这样的帮助。紧接着2026年的当下，我继续参与了天津地区招生，这次参与了绝大多数的线下场次，也因此收获了更多的经验和内容。鉴于我认为我对学校各个方面还算了解，也算是小六边形，便动笔写了一份1.5万字的《宁诺天津FAQ》，但当时只在我的同乡群和我的朋友们中宣传。由于文字较为粗糙，很多地方措辞并不“官方”，我特意嘱咐了他们当个乐看就好，很多内容不要大张旗鼓的宣传。事实上，这份《FAQ》名有“天津”，实则无太多的地域特色，对UNNC 本科新生都普遍适用。

2026年暑期科研时，我的一位朋友，他当时负责浙江大学本科新生指引一部分的更新，向我提到了这个著作，我受到了极大程度的震撼，震撼于 @好神奇 前辈能有如此大的毅力把这件事情做了四年还能传承下去。加之与两年新生接触时，发现他们问的问题很多时候都是类似的，我觉得需要一个“媒介”能够传递这些信息，既然受 ZJU指引 的启发，不如就叫UNNC 指引吧。

我当然希望在未来依然能够有人能够继续这项“事业”，坚持五年、十年、二十年甚至更久。我也希望《指引》的修订未来能够坚持开源共享、群策群力，在一届又一届的接力传承中，建立起于众届师兄师姐们智慧之上的城市。

《指引》不是一本“万能手册”，“到宁诺来做什么”，这个问题我无法给予大家回答。但我会尽我所能，告诉大家“宁诺有什么”“到宁诺来能做什么”。而至于“将来毕业后成为什么样的人”，需要诸位在四年的摸爬滚打中不断探索，找到属于自己的答案。
