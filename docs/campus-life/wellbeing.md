# 医疗与心理支持

学校身心健康中心位于生活区 **23 号楼一楼**，包括医务室和心理咨询服务。

## 医务室

医务室可以处理常见疾病和轻微外伤，并提供基础诊疗、急救、用药指导、转诊预约、疫苗咨询及部分医疗器械借用等服务。如果需要特定药品，可以提前通过邮件询问是否有库存。

- **地点：** 生活区 23 号楼一楼
- **日常门诊：** 周一至周日 8:30–22:30
- **24 小时医疗服务热线：** [0574-88180120](tel:057488180120)
- **24 小时紧急救援专线：** [15168185997](tel:15168185997)
- **邮箱：** [UNNC.Clinic@nottingham.edu.cn](mailto:UNNC.Clinic@nottingham.edu.cn)

[查看医务室官方介绍](https://www.nottingham.edu.cn/cn/health-and-wellbeing-centre/%E5%8C%BB%E5%8A%A1%E5%AE%A4/%E5%8C%BB%E5%8A%A1%E5%AE%A4.aspx){ .resource-link }

!!! danger "紧急情况"
    如遇危及生命的紧急情况，请立即拨打 **120**，同时联系校园医疗紧急救援专线或校园安保，以便急救人员进入校园并尽快找到患者。

## 心理咨询

University Counselling Service 提供专业心理咨询。适应大学生活、学业压力、焦虑或低落、失眠、家庭与人际关系等问题，都可以成为寻求咨询的理由，不需要等到问题非常严重才预约。（你们一定不要从高空肘击地面口牙！）

- **地点：** 生活区 23 号楼一楼后门，128–135 室
- **服务时间：** 周一至周五 8:30–12:00、13:30–17:00；周末及公共假期不开放预约
- **预约入口：** [心理咨询预约系统](https://counsel.nottingham.edu.cn/){ .resource-link }，校外访问时也可通过 Student Portal、AppGateway 或 UNNC VPN 进入
- **邮箱：** [Universitycounsellingservice@nottingham.edu.cn](mailto:Universitycounsellingservice@nottingham.edu.cn)

第一次咨询主要用于说明近况，并与咨询师讨论后续是否需要继续咨询。学期中的需求高峰可能需要等待，建议有需要时尽早预约。

[查看心理咨询预约指南](https://www.nottingham.edu.cn/en/health-and-wellbeing-centre/university-counselling-service/appointments.aspx){ .resource-link }

!!! warning "需要立即帮助时"
    心理咨询服务不提供夜间、周末或公共假期预约。如果你或身边的人有立即伤害自己或他人的风险，请不要独处，立即联系 **120 / 110**、校园安保或医务室 24 小时热线，并尽快让可信任的同学、老师或家人陪同。
