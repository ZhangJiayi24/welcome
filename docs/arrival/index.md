# 抵达宁波

## 宁波交通枢纽

### 宁波栎社国际机场

宁波市目前唯一的民用机场，全国到宁波的所有飞机在此降落。距离学校搭乘地铁约1小时5分钟，花费4元；打车约20-25分钟，花费约25-30元。

### 宁波站

宁波市客流量最大的火车站，大部分G/D字头动车均在此停靠。距离学校搭乘地铁约40分钟，花费4元；打车约30-40分钟，花费约25-30元。

乘坐火车或动车组列车到校报到的同学，可以直接通过铁路12306线上购买学生票，携带录取通知书核验学生身份，具体流程见[铁路12306的这篇推文](https://mp.weixin.qq.com/s/DtQizL0r4CqfC4onefS-sg){ .resource-link }。在领取学生证之后，后续乘坐火车或动车组列车往返学校和家庭所在地，可以直接通过12306线上购票，使用学生证核验学生身份，无需另外操作。

## 入校

乘坐火车和飞机前往宁波的同学，建议乘坐宁波地铁到 8 号线地铁学府路站 B 口出站，然后直走约 200 米，路口有一块写着学校名字的石头可以合影，还可以拍一张钟楼的照片；再前行约 700 米即到学校 3 号门，进入后即是学校生活区。

高考新生与家长凭录取通知书进校。报到当天请根据出行方式从指定校门进入：

- **步行进校：** 高考新生与家长可从 3 号门或 4 号门步行进校。
- **车辆进校：** 校内机动车单向行驶，从 1 号门进入、4 号门驶出。
- **机动车停放：** 请听从现场安保人员指挥，有序停放车辆。校内停车位有限；如校内车位已满，请将车辆停放在校外泰康东路沿线。
- **校园限速：** 校园内车辆限速 20 km/h，请驾车进校的学生及家长注意行车安全。

## 家长入校预约

报到当天，高考新生与家长按上述方式凭录取通知书入校。非报到日期如家长需要入校，请提前确认学校当日的访客规定，并按门岗要求登记或预约。
