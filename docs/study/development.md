# 科研与素质教育

宁诺的本科培养不只包括课程和考试。学校、三大学院及教育发展基金会还提供科研、竞赛、行业实践、创新创业、公益和体育等课外项目。它们大多属于第二课堂或协同课程，是否计入诺丁汉卓越奖（NAA）、SPDPO 学分或其他官方记录，以及当学年是否开放，应以 Moodle、学生门户、校内邮件和项目负责部门的通知为准。

## FURP / 本科生科研实习项目

FURP 是宁诺本科科研训练的重要入口。学生可以在导师指导下参与真实课题，通过阅读文献、提出问题、实验或调研、采集和分析数据、撰写报告等环节，体验科研的完整流程。

- **主要面向：** 理工学院全体本科生。大一学生可选择的项目相对较少，主要科研项目面向大二及以上学生。
- **项目形式：** 目前主要为校内导师课题，可在学期中或暑期开展。
- **项目发布：** 通常在 11 月中下旬通过校内邮件通知。附件 Excel 会列明项目负责老师、研究主题、项目简介以及申请者需要具备的技能等信息。
- **信息渠道：** FURP 属于理工学院项目，招募邮件一般不会同步发送给其他学院。理工学院学生应重点留意学院邮件；其他学院目前没有对应的公开公众号渠道。
- **适合人群：** 想确认研究兴趣、准备硕博申请或积累科研经历的学生。

!!! tip "神秘小纸条"
    其实，如果你自己的能力真的特别强，又很想做科研，不一定非要等项目招募。你可以直接去学校官网搜索老师的研究方向，看看谁的方向最让你感兴趣，再给老师发邮件“套磁”。动笔前记得先看[邮件礼仪](email-etiquette.md){ .resource-link }。

## SEED 学生实践与职业发展计划

SEED 是诺丁汉大学商学院（中国）的旗舰课外培养体系。它不是单一竞赛或科研项目，而是把四类成长路径放在同一个框架中：

- **研究探索：** 从研究方法训练进入导师课题，以研究助理身份参与金融、数据分析、市场营销、创业与创新等方向的研究。
- **行业实践：** 包括全球导师、X-Culture、学生咨询、国际创业和数字化转型咨询等项目，部分项目采用选拔制并直接对接企业。
- **商业竞赛：** 提供赛前工作坊、赛中导师指导、赛后复盘以及资源或资金支持，覆盖金融、商业创新、可持续发展和创业赛事。
- **社会影响：** 围绕社区、环保和可持续议题设计项目，训练项目设计、利益相关方沟通和成效评估能力。

SEED 以商学院学生为主要服务对象，但部分活动或奖学金可能面向全校开放。2024/25 学年的研究项目公开提供了 80 余个研究助理岗位，竞赛项目支持了 300 余名学生。

**注意信息差：** SEED 的项目邮件和招募信息主要在商学院内部流转，通常不会同步发送给其他学院。非商学院学生如果希望参与，应主动关注 SEED 公众号，并多与商学院同学交流，及时了解开放项目和申请时间。

## 未来菁英计划

人文社科学院的长期成长项目，整合科研助理、年度学术会议、学术分享、竞赛支持、企业参访、实习与志愿者机会、职业工作坊和学生领导力岗位。

对于晋级更高阶段比赛的学生，学院公开说明可提供专项训练及相关费用支持。2024/25 学年有 500 余名学生参与相关活动。

[查看未来菁英计划](https://www.nottingham.edu.cn/cn/humanities-and-social-sciences/career-development/career-development.aspx){ .resource-link }

## 诺丁汉卓越奖 NAA

诺丁汉卓越奖（Nottingham Advantage Award）是学校的就业力与课外能力培养框架。学生通过额外课程和协同项目发展个人、学术及职业能力，并学习如何向雇主展示这些经历。

具体模块可由学院、就业指导办公室、图书馆或孵化园承办，往年内容包括咨询、创业、学生治理和数据素养等。每年实际开放模块以 Moodle 为准。

[查看 NAA 介绍](https://www.nottingham.edu.cn/en/careers/naa/nottingham-advantage-award.aspx){ .resource-link }

## 学生竞赛发展基金

这是面向全校的竞赛支持平台，不是一场具体比赛。项目从赛事政策、奖励设置、行政协调和学术资源等方面支持学生及学院开展竞赛。参加比赛前，可同时留意学院项目、学生事务与发展中心和教育发展基金会的通知，确认培训、报名费、差旅费或获奖奖励是否有对应支持渠道。

[查看学生竞赛发展基金](https://www.nottingham.edu.cn/cn/foundation/donation-projects/student-activity.aspx){ .resource-link }

## 李达三创新创业体系

如果你希望把想法继续发展成原型、创业项目或公司，可以关注李达三孵化园及其相关项目：

- **创客实验室：** 面向全校学生，通过开放空间和参与式活动训练创新与创业思维。
- **创新工作室：** 支持跨学院学生围绕企业和社会问题开展实践，提供空间、设备、导师和企业合作资源。
- **李达三创新基金：** 支持符合条件的师生将知识和科研成果从 0 推进到 1；公开材料称每年资助额度至少 300 万元。
- **孵化服务：** 从创意、商业实践到企业孵化，对接学校科研设施、产业伙伴和创业网络。

[查看李达三孵化园](https://www.nottingham.edu.cn/cn/research-and-knowledge-exchange/li-dak-sum-incubator/li-dak-sum-incubator.aspx){ .resource-link } · [查看学生创新项目](https://www.nottingham.edu.cn/cn/foundation/donation-projects/student-activity.aspx){ .resource-link }

## 公益实践与体育竞赛

“点亮未来”公益志愿服务由暑期社会实践发展而来，覆盖支教、赛会志愿、动植物保护研究和传统文化研习等方向。

体育方面，EXCEL 是校代表队竞技通道，覆盖篮球、足球、羽毛球、击剑、赛艇、攀岩、排球等项目，通常在每学期初进行选拔。

[点亮未来](https://www.nottingham.edu.cn/cn/department-of-campus-life/student-engagement/%E5%85%AC%E7%9B%8A%E5%BF%97%E6%84%BF%E6%9C%8D%E5%8A%A1%E9%A1%B9%E7%9B%AE.aspx){ .resource-link } · [体育 EXCEL](https://www.nottingham.edu.cn/en/sport/excel/excel.aspx){ .resource-link }

## 奖助

学校教育发展基金会还设有多种与课外发展相关的奖励或资助，例如梦想奖学金、校长推荐优秀毕业生奖、实习金像奖、乐歌奖学金、领创人才奖学金等，分别关注科技创新、实践、公益、体育、文艺、实习表现或创业领导力。

这些项目的开放年份、对象和评选条件并不完全相同，不能把项目汇总页视为当年的申请清单。申请前应查看基金会或负责部门发布的最新通知。
