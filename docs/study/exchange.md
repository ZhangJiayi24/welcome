# Exchange

Exchange（海外交换）通常为一学期或一年。

Exchange 和 2+2 并不是一回事。2+2 是完整的学制选择，学生后两年在英国诺丁汉大学学习；Exchange 结束后会返回宁诺继续完成学业。

## 申请前需要关注什么

!!! warning "以当年通知为准"
    学校一般会在大二第一学期末发邮件通知申请。项目、名额、截止日期和官网信息都可能调整，请以 GEO、学院邮件和申请时的官方网页为准。

- **院校与名额：** 不同专业开放的交换院校和 quota 都不一样。前往官方 [Before applying](https://www.nottingham.edu.cn/en/global/exchange-and-study-abroad/outbound/before-applying.aspx){ .resource-link } 页面，在页面底部查看当年的 **Quota form**。
- **成绩要求：** 申请通常参考大二第一学期的成绩，部分合作院校还可能有语言成绩或先修课程要求。
- **课程匹配：** 申请前应与学院确认海外课程能否完成学分转换，避免影响后续选课和毕业进度。
- **费用安排：** Exchange 是向宁诺缴纳 11 万元学费，不需要再向对方院校缴纳学费；Study Abroad 则需要向对方院校缴纳学费。住宿、交通、签证、保险和生活费等支出另行计算。

学校官方页面提供申请前准备、获得提名后的流程、合作院校名单和往届学生经历等信息：

[查看 Outbound Exchange 官方页面](https://www.nottingham.edu.cn/en/global/exchange-and-study-abroad/outbound/outbound-exchange.aspx){ .resource-link }
