# 2026年 UNNC 本科新生指南

基于 Material for MkDocs 的非官方学生指南框架。

## 本地预览

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
mkdocs serve
```

浏览器访问 `http://127.0.0.1:8000/`。

## 线上地址

站点计划发布至 `https://welcome.unncers.com/`，由 GitHub Actions 自动构建并部署至 GitHub Pages。

## 构建检查

```bash
mkdocs build --strict
```

> 当前仅为栏目与视觉框架。正文内容尚未审校，不代表宁波诺丁汉大学官方信息。
