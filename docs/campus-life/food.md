# 餐饮

校园内的餐饮选择比较丰富，从十元左右的基础餐食到五六十元的正餐都有。整体口味正常、略偏清淡，大多数同学不需要担心完全吃不习惯。

## 四个食堂

- **一食堂 / 万诺餐厅：** 形式与常见的高中或大学食堂相近，可以直接选择已经做好的餐食。
- **二食堂 / 宁诺味道：** 以档口点餐为主，点单后等待叫号取餐。
- **三食堂 / 诺兰餐厅：** 以档口点餐为主，点单后等待叫号取餐。
- **四食堂 / 第三空间：** 同样采用点餐、叫号取餐的形式。

食堂的常用叫法和所在位置可查看[黑话 → 觅食地点](../toolkit/glossary.md#campus-food){ .resource-link }。

## 其他餐饮

除四个食堂外，校园内目前还有小灶台、东城食堂和清真食堂，以及 KFC、麦当劳、必胜客、巴黎贝甜等连锁餐饮。钟楼内还有学校西餐厅阿兰碧卡。

校内奶茶和咖啡选择也很多，包括蜜雪冰城、瑞幸、星巴克、Tims、霸王茶姬、CoCo，以及学校自己的咖啡店品牌 TRIO 等。部分品牌在校园内有不止一家门店。

!!! note "店铺会变化"
    校内商户可能随合同和学年调整，具体营业门店、位置和开放时间以现场信息为准。

## 外卖

如果临时不想在校内用餐，也可以点外卖。大多数外卖可以送到宿舍楼附近的指定取餐点；实际配送范围和楼栋管理要求以当时安排为准。
